# ZopfliCompress

使用 Zopfli 对基于 Deflate 算法的文件进行重压缩。\
Uses Zopfli to recompress files based on the Deflate algorithm.

目前，它支持对 PNG 文件中的 IDAT 数据块进行重压缩。未来将支持对 PNG 文件中的 iCCP、zTXt、iTXt 和 fdAT 数据块进行重压缩。\
Currently, it supports recompressing IDAT chunks within PNG files. Future updates will add support for recompressing iCCP, zTXt, iTXt, and fdAT chunks in PNG files.

此外，还可能支持其他格式，例如 ZIP 等。\
Additionally, support for other formats—such as ZIP archives—may be added in the future.
